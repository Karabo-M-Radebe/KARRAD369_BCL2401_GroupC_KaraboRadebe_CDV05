Gaurdians' Playlists

A playlist generating project

In this challenge, I combined your knowledge of JavaScript arrays and the map() function to create a personalized "Awesome Mix" playlist for each of the Guardians of the Galaxy.

I had to work with the map() functions in order to create a playlist generator based on genres of the songs. Using this function, along with the filter() function I had to assign preferences and generate a playlist.

Then the next object was to display the playlist by appending certain elements.
